利用𝑯𝒆𝒙𝒐+𝑮𝒊𝒕𝒉𝒖𝒃搭建属于自己的博客(Windows)

注：

  1. 文件_𝒸𝑜𝓃𝒻𝒾𝑔.𝓎𝓂𝓁非常重要！
  2. 在安装操作过程中，出现无法处理的错误可以将建立的博客文件夹删除。
  3. 常用命令：
    • hexo init #初始化博客
    • hexo new 文章名称 #创建文章(简写：hexo n)
    • hexo clean #清理
    • hexo generate #生成(简写:hexo g)
    • hexo server #启动本地博客服务(简写：hexo s)

一、安装前准备

  1. Node.js (版本建议Node.js12.0及以上)
    【包括两个组件：Node.js 和 npm包管理器】
  2. Git

二、本地启动

  • 1.打开cmd命令行

    查看Node.js和npm的版本：

  • 2.安装Hexo博客框架

    由于国内利用npm安装会比较慢，所以需要使用cnpm来安装淘宝的镜像源。

  • 3.使用Hexo搭建博客

    首先创建一个存放博客的文件夹，并进入到文件夹内。

    报错，需要安装git

    可以去博客目录下，查看生成的博客的基础框架内容。使用hexo s命令启动服务器。

    将链接复制到浏览器上即可访问博客内容。

二、远端访问部署(Github)

  • 1.在github下创建新的仓库。

    用户部署个人博客的Github仓库命名必须符合：”Github账号的昵称.github.io”

  • 2.回到命令行下部署插件

  • 3.配置_config.yml

    在博客安装目录下打开编辑_config.yml，到文件最底部，进行配置。

    repo的网址是在Github上床架的仓库形成的地址，直接复制过来就好。
